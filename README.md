# Barbearia-Vision-Men
Introdução ao Projeto

Este projeto consiste no desenvolvimento de uma landing page simples para um comércio local fictício, com o objetivo de consolidar os conhecimentos iniciais em HTML e CSS.
A página foi estruturada seguindo o mesmo padrão do projeto Site para Imobiliária, aplicando boas práticas de organização, semântica e estilização.

A landing page é composta por cinco seções principais — Topo, Área Principal, Serviços, Sobre Nós e Contato — incluindo navegação interna, logomarca, imagens ilustrativas, 
alternância de cores entre seções e integração com o Google Maps. O projeto simula um negócio com atendimento ao público e possibilidade de agendamento, 
demonstrando a capacidade de criar uma interface funcional, visualmente organizada e adequada para apresentação de um comércio local.
